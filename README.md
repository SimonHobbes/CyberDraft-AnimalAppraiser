# 🐴 牛马的自我养成 (Cyber Draft-Animal Appraiser)

**一款专为当代打工人打造的跨地区、高自由度薪资与五险一金双向推演工具。**

## ✨ 项目愿景与解决的痛点

- **打破 HR 的薪资信息差**：“到底交多少比例的公积金？”各项明细数据和低基数缴纳影响，在这里一目了然。
- **解决跨城市 offer 比较难的问题**：一键导入不同城市的五险一金模板，横向对比各地实发工资和企业缴纳成本。
- **告别繁琐的本地保存**：实现**“一个 URL 传递你的赛博身价”**。所有的薪资、五险一金参数均可通过 `lz-string` 压缩生成极其整洁的短链接，随时发送给朋友，点击即可 100% 还原计算现场。

## 🛠 核心技术栈

- ⚡ **Vite**: 极速的现代前端构建工具。
- ⚛️ **React**: 配合 Hooks 精准管理复杂的表单联动与状态计算。
- 🎨 **DaisyUI + TailwindCSS**: 提供开箱即用的暗黑风格/赛博朋克主题组件，无需沉重的 JS 互操作。
- 🔗 **lz-string**: 将庞大的地区比例、基数和薪资参数压缩处理，转换成 URL 安全的 Base64，突破传统 Query 参数长度限制。

## 🚀 快速开始

### 本地运行

1. 克隆项目并进入目录：
```bash
git clone https://github.com/SimonHobbes/CyberDraft-AnimalAppraiser.git
cd CyberDraft-AnimalAppraiser
```

2. 安装依赖包：
```bash
npm install
```

3. 启动本地开发服务：
```bash
npm run dev
```
打开浏览器访问 `http://localhost:5173` 即可开始使用。

## 📦 自动构建与部署

项目已内置了 **GitHub Actions** 自动化工作流。

只要你将代码 Push 到 GitHub 仓库的主分支 (`main` 或 `master`)，后台将会自动打包 (`npm run build`) 并将静态资源部署到 **GitHub Pages**。

> **⚠️ 注意**：如果你修改了仓库名，请进入 `.github/workflows/deploy.yml` 文件内，将 `npm run build -- --base=/CyberDraft-AnimalAppraiser/` 后的 `--base` 参数修改为你的实际仓库名。若使用独立域名或用户名主页部署，可将 `--base` 删去。

## 📁 核心功能概览

- **薪资双向推演**：输入应发工资即可推算各项扣费及实发工资；左右拖动“缴纳基数”滑块，系统将自动核算低基数缴费下的**企业节省成本(差额)**。
- **图表可视化展示**：主页含清晰的饼状图展示“实发工资、个人缴纳、企业缴纳、低基数差额”的具体划分比例。
- **模板配置与管理**：系统自带基础通用模板，你可以在 `配置设置` 中调整养老、医疗、失业、公积金等各项的详细比例与上下限，并支持导出为 JSON 文件本地保存。
- **移动端全适配**：全面适配了手机端的各类尺寸（如 iPhone 等竖屏设备），保证在手机上也能极其舒适地通过滑块调节薪资。
