# 🐴 牛马的自我养成 (Cyber Draft-Animal Appraiser)

以图表的形式展示五险一金的缴纳情况，并且支持定制模板适应各地需求，且支持一键 URL 分享你的配置与结果, 拒绝漏缴少缴

## 📁 模板配置指南 (欢迎提交 PR！)

本项目所有的城市五险一金配置模板均存放在 `public/configs/` 目录下（格式为 `.json` 文件）。各地的社保公积金政策差异极大，我们**非常欢迎大家导出自己所在城市的精确配置，并提交 Pull Request** 补充到项目中，帮助更多打工人！

### 🎈 如何贡献模板？
1. 在网页端通过 **⚙️ 设置** 调整好你所在或者目标城市的五险一金精确比例及基数上下限。
2. 点击顶部的 **📤 导出** 按钮，将配置下载为 `.json` 模板文件。
3. 将文件重命名为年份+城市拼音（如 `2026_shanghai.json`），放入代码仓库的 `public/configs/` 目录中。
4. 提交 Pull Request

### 📜 模板字段含义说明

一个标准的 `.json` 配置模板包含如下结构：

```json
{
  "name": "上海上半年",         // 必填：该配置的唯一标识
  "year": 2024,               // 可选：该政策规则适用的年份
  "city": "上海",             // 可选：界面的下拉列表中显示的中文城市名称
  
  "socialBase": {             // 社保缴纳基数限制
    "min": 7384,              // 社保缴费基数底线（当地平均工资的60%）
    "max": 36921              // 社保缴费基数顶线（当地平均工资的300%）
  },
  "fundBase": {               // 住房公积金缴纳基数限制
    "min": 2690,              // 公积金缴费基数下限（通常为当地最低工资）
    "max": 36921              // 公积金缴费基数上限
  },
  
  "fundRatioRange": [         // 该地区合规的公积金缴纳比例可选范围
    5, 6, 7, 8, 10, 12        // 用户只能在这些固定阶梯自然数中滑动选择公积金比例
  ],
  
  "items": {                  // 各个具体险种的扣缴比例
    "pension": {              // 【养老保险】
      "individualRate": 8,    // 个人缴纳比例 (填写数字，8 即代表 8%)
      "companyRate": 16       // 企业缴纳比例
    },
    "medical": {              // 【医疗保险】
      "individualRate": 2,
      "companyRate": 10.5
    },
    "unemployment": {         // 【失业保险】
      "individualRate": 0.5,
      "companyRate": 0.5
    },
    "injury": {               // 【工伤保险】(一般仅企业缴纳，各地通常浮动)
      "individualRate": 0,
      "companyRate": 0.26
    },
    "maternity": {            // 【生育保险】(一般仅企业缴纳。部分城市已与医保合并，视情况填 0 或拆分)
      "individualRate": 0,
      "companyRate": 1
    },
    "housingFund": {          // 【住房公积金】
      "individualRate": 7,    // 个人公积金比例（此数值为配置载入时滑块的“默认位置”）
      "companyRate": 7        // 企业公积金比例
    }
  }
}
```
